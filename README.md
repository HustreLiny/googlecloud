**不要点激活或者升级**<br>
我们操作完成前边七步，进入谷歌云平台，你会看到下图红框中的这句话，千万不要好奇点激活，否则免费结束后，继续使用就会被扣费。在概览页面也不要点击升级按钮。<br>
